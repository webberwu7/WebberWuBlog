# webber's note

使用 hugo 建立靜態網站並發佈於 Github Page

網站連結 [webber筆記本](https://webberwu7.github.io/WebberWuBlog/)


## 使用方法
### Run Local Hugo Site
```shell
hugo server -D 
```

### add new page
```shell
hugo new posts/{title_name}.md
```


### Build static web page
```shell
hugo
```

### Update submodule
```shell
git submodule update --remote
```
